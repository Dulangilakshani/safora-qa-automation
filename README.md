# Safora QA Engineering Intern Assignment

This repository contains the automated UI test suite for the **Contact Us** form on the Safora website, implemented using the **Playwright** framework.

## Project Overview
- **Target Environment:** https://safora.se/en/
- **Automation Tool:** Playwright (JavaScript)
- **Scenarios Covered:** 
  - Form validation errors on empty submission.
  - Successful form submission with valid inputs.

##  Prerequisites
Before running the tests, ensure you have the following installed on your local machine:
- [Node.js](https://nodejs.org/) (v14 or higher recommended)
